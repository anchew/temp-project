# Network Attack Project

Run:

```bash
python3 code/main.py
```

The code reads the Excel file from `Network Attack` and puts the output in `output`.

Useful output:

- `dataset_notes.txt`
- `graph_notes.txt`
- `model_notes.txt`
- `model_results.csv`
- `model_reports.txt`
- `graphs/` for the PNG graph files
